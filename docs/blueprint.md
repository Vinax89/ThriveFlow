# **App Name**: ThriveFlow

## Core Features:

- Budget Tracking: Allows users to track their income and expenses.
- Debt Management: Provides tools for users to manage and pay off their debts.
- Financial Goal Setting: Enables users to set and track progress toward financial goals.
- BNPL Tracking: Tracks Buy Now, Pay Later plans.
- Cash Flow Projections: Predicts future cash flow based on current income, expenses, and debts. It can access relevant financial data to inform the prediction, acting as a tool.

## Style Guidelines:

- Primary color: Deep calming blue (#29ABE2) to instill trust and security.
- Background color: Very light desaturated blue (#E5F6FD) for a clean, calming backdrop.
- Accent color: Soft green (#8DD3BB) for positive reinforcement and calls to action.
- Font: 'Inter', sans-serif, for both headings and body text to create a modern and readable interface.
- Use consistent, clean line icons to represent financial categories and actions.
- Maintain a clean, organized layout with clear visual hierarchy to prioritize important information.
- Incorporate subtle transitions and animations to enhance user experience without being distracting.
